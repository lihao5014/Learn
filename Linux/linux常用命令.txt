2.linux软件安装与卸载：
	sudo apt-get update                      //更新软件安装源列表：
	sudo apt install 软件名
	sudo apt-get install 软件名
	dpkg --list                              //显示的是你电脑上安装的所有软件。
	sudo apt-get --purge remove 软件名       //彻底删除软件，包括配置文件一起删除
	sudo apt-get remove 软件名               //删除软件时保留配置文件
	sudo apt autoremove                      //清空卸载残留

5.使用git统计代码行数命令：
（1）git log --author="username" start_commit_id..end_commit_id --pretty=tformat: --numstat | gawk '{ add += $1 ; subs += $2 ; loc += $1 - $2 } END { printf "added lines: %s removed lines : %s total lines: %s\n",add,subs,loc }'
（2）将'start_commit_id'改为第一次提交的id，'end_commit_id'改为最后一次提交的id，'username'改为git用户名。就可以统计出两次提交间代码的变化行数。

7.ubuntu清除缓存以及不用的安装包
（1）清理旧版本的软件缓存：
	 sudo apt-get autoclean
（2）清理所有软件缓存：
	 sudo apt-get clean
（3）删除系统不再使用的孤立软件
	 sudo apt-get autoremove
（4）不要轻易使用aptitude命令，他有可能会卸载apt-get
	(X)sudo apt-get install aptitude

8.Ubuntu和Windows直接共享文件夹：
（1）Ubuntu上安装VMware Tools工具。
（2）VMware虚拟机设置 -> 选项Tab -> 共享文件夹 -> 添加一个共享文件夹― -> 总是启用 -> 确定。
（3）可进入设置的共享目录： cd /mnt/hgfs/share/
（4）在主目录下创建共享文件夹的软连接（相当于快捷方式）：ln -s /mnt/hgfs/share/ ~/

9.查询磁盘空间：
（1）df -hl                                #查看当前目录的磁盘占用
（2）du -sh 文件名                         #查看某一文件的磁盘占用
（3）sudo du -sh ./* --exclude proc        #查看某一目录下所有文件夹的磁盘占用

13.Linux中配置第三方库：
（1）sudo mkdir /usr/local/arm-gcc
（2）sudo cp -f sysroot-glibc-linaro-2.25-2019.12-aarch64-linux-gnu.tar /usr/local/arm-gcc
（3）sudo tar -vxf /usr/local/arm-gcc/sysroot-glibc-linaro-2.25-2019.12-aarch64-linux-gnu.tar
（4）sudo nano ~/.bashrc
（5）export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/usr/local/arm-gcc/sysroot-glibc-linaro-2.25-2019.12-aarch64-linux-gnu/lib
（6）source ~/.bashrc
